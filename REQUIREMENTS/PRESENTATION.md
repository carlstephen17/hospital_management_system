# Presentation Guide

## Demo Overview
This presentation demonstrates the Hospital Management System built with React/Vite frontend and Node.js/MySQL backend.

## Slide Plan

1. **Introduction**
   - Project title: Hospital Management System
   - Course: Database Management Systems 2
   - Goal: CRUD application with advanced SQL reports and relational algebra mapping.

2. **System Architecture**
   - Frontend: React + Vite
   - Backend: Node.js + Express + MySQL
   - Database: patients, doctors, treatments, appointments, bills, patient_treatments

3. **User Interface**
   - Sidebar navigation
   - Patient, doctor, treatment, appointment, and bill management pages
   - Report page

4. **CRUD Demonstration**
   - Show creating, reading, editing, and deleting a record
   - Example: add a patient, create an appointment, generate a bill

5. **Reports**
   - Load Cartesian product report
   - Show UNION report
   - Show SET DIFFERENCE report
   - Explain why these queries matter

6. **Relational Algebra Mapping**
   - Present SQL ↔ relational algebra examples
   - Highlight symbols: `σ`, `π`, `×`, `∪`, `−`

7. **Conclusion**
   - Summarize functionality
   - Mention documentation and deliverables
   - Next steps for improvement

## Demo Script
- Start the app and show the home page.
- Navigate through the sidebar to each module.
- Create a patient and doctor record.
- Create an appointment and bill.
- Open the reports page and click `Load Reports`.
- Describe the outputs and how they correspond to relational algebra.

## Timing
- Aim for 10–15 minutes.
- Spend roughly 1–2 minutes per major section.
- Reserve 2 minutes to answer questions or show extra functionality.
